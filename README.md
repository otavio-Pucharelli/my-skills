# 📚 Coletânea de Skills

Bem-vindo ao repositório central de **Skills**! 

Este diretório contém um conjunto de instruções e perfis especializados (skills) projetados para expandir e focar as capacidades do assistente em áreas específicas de desenvolvimento de software, engenharia, segurança e testes.

---

## 🏗️ Estrutura de Skills Disponíveis

Aqui estão as skills atualmente mapeadas e prontas para uso:

### ☕ Java Backend
Focadas no ecossistema de desenvolvimento Java (Spring Boot, Quarkus) e arquitetura de backend.

* **[SEO-Audit](./Java-skills/SEO-Audit/Skill.md)**
  * **Descrição:** Especialista em Auditoria Técnica e SEO. Analisa a infraestrutura e código em busca de melhorias de performance (Vitals), Server-Side Rendering (SSR) e metadados.
* **[Java Code Security Checker](./Java-skills/Security-checker/Skill.md)**
  * **Descrição:** Realiza validações de segurança ativas e procura por vulnerabilidades (OWASP Top 10, Injeções, Configurações Inseguras) em códigos Java, com foco especializado em Spring Boot e Quarkus.
* **[Java Senior Developer](./Java-skills/java-senior-developer/Skill.md)**
  * **Descrição:** Atua como um Desenvolvedor Java Web Sênior, especialista em Spring Boot e Quarkus. Focado em alinhar boas práticas (Clean Architecture), obrigatoriedade de testes automatizados (TDD) e exigir aprovações parciais durante a implementação.

### 🎨 Frontend
Focadas no ecossistema e visualização de aplicações Client-side.

* **[Frontend Security Checker](./Front-Skill/Front-security-checker/Skill.md)**
  * **Descrição:** Examina detalhadamente componentes visuais, lógicas em JS/TS, consumo de APIs e configurações de frameworks (React, Angular, Vue) para identificar falhas da camada do cliente (XSS, Auth Tokens, IDOR).

### 🧪 Testes e Qualidade
Focadas em garantia de qualidade e metodologias guiadas por testes.

* **[Java TDD Test Creator](./Test-creator/SKILL.md)**
  * **Descrição:** Atua como um parceiro interativo de Test-Driven Development (TDD) para Java, desenhando cenários, mapeando edge-cases e gerando testes (Red phase) antes da implementação em si.

### ⚙️ Utilitários
* **[Base Skill](./base-skill.md)**
  * **Descrição:** Template inicial ou estruturação base para a criação de novas skills personalizadas.

---

## 🚀 Como Utilizar

Sempre que precisar de um contexto de análise aprofundado, basta referenciar a skill correspondente (`@Skill.md`) no chat do assistente ou garantir que a skill esteja adicionada à janela de conversação do Gemini antes de iniciar sua análise ou escrita de código.
